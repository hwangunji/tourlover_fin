# tourlover_fin
휠체어 사용자를 위한 길안내 어플
이 어플은 교내 지도를 분석하여 길의 경사에 따라 다른 색으로 표시한 후 휠체어가 지나다닐 수 있는 길로 안내합니다. 또 교내 건물의 장애인 시설안내 기능, 도로 위 돌발 상황 추가기능, 음성안내 기능이 있습니다.
